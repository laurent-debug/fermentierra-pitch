# fermentierra-pitch

Présentation interactive moderne pour la prospection et les rendez-vous clients (food business).

## Aperçu
- Application web statique responsive (HTML/CSS/JS)
- Adaptée à un usage sur laptop, tablette ou partage email
- Sections : À propos, Avantages, Segments, Gammes, Services, Contact
- Palette : vert forêt, orange doux, beige naturel

## Mise en ligne
1. Cloner ce repo ou télécharger le .zip
2. Dézipper le contenu dans un dossier local
3. Ouvrir `index.html` pour prévisualiser en local
4. Publier sur GitHub Pages :
   - Aller dans Settings > Pages > Source : main (/) 
   - Accéder au lien public généré (ex: https://username.github.io/fermentierra-pitch/)
5. (Facultatif) Ajouter un fichier `CNAME` pour connecter un domaine personnalisé

## Déployer sur votre propre hébergement
- Uploader tous les fichiers via FTP/File Manager à la racine ou dans un dossier spécifique
- Accéder ensuite en ligne (ex: www.mon-domaine.com/pitch/)

## Modifier le design/contenu
- Texte : ouvrez `index.html` avec VS Code/Sublime Text
- Styling : modifiez `styles.css`
- Images : remplacez/ajoutez dans `/assets` ou `/img`
